링크 파일
  - 어떤 파일을 가리키는 파일

 - 하드 링크 : 원본 파일의 물리적 주소(i-node) 값을 가리키는 링크
 - 심볼릭 링크 : 원본 파일의 논리적 경로를 가리키는 링크

 - 하드 링크
      $ ln  원본파일    링크파일

 - 심볼릭 링크
      $ ln  -s  원본파일   링크파일

Ch05. 파일 및 디렉터리 검색


grep  -  파일에서 특정한 패턴을 찾는 명령어

grep  [option]  PATTERN  FILE

특정 패턴이 존재하는 행 찾기
 $ grep  PATTERN  FILE

특정 단어가 존재하는 행 찾기
 $ grep  -w  PATTERN  FILE

특정 패턴이 존재하는 행을 행 번호와 함께 출력
 $ grep   -n  PATTERN  FILE

특정 패턴이 존재하는 행을 배제하고 출력
 $ grep   -v  PATTERN  FILE
