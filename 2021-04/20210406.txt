Ch08. 쉘 명령어 사용

쉘 메타문자
(1) 경로 이름 메타 문자
 1)  ~  : 로그인 한 사용자의 홈 디렉터리
 

(2) 파일 이름 대체 메타 문자
 1)  *  : 아무것도 없거나 하나 이상의 임의의 문자를 대체하는 메타 문자
 2)  ?  : 임의의 한 글자
 3)  []  : 문자 세트로 정의된 범위 내의 한 글자


(3) 인용부호 메타 문자
 1) ' '  : 홑 따옴표 안에 있는 메타 문자를 일반 문자로 처리하는 인용부호 메타 문자
 2) " " : 이중 따옴표 안에 있는 메타 문자 중 ' $ ',  ' ` ',  ' \ ' 를 제외한 나머지를 일반 문자로 처리하는 인용부호 메타 문자
 3) ` `  : 역 홑 따옴표 안에 있는 내용의 실행 결과로 대체하는 인용부호 메타 문자


(4) 방향재지정 메타문자

<표준 입출력>
표준 입력(Standard Input, stdin, 0)    - 키보드
표준 출력(Standard Output, stdout, 1)  - 모니터
표준 에러(Standard Error, stderr, 2)     - 모니터

방향재지정(Redirection) - 입 출력 방향을 기본 장치가 아닌 다른 곳으로 지정하는 것

 >
>>
 |

(1) 표준 출력 재지정
   command  >  FILE
   command >> FILE


(2) 표준 에러 출력 재지정
   command   2>   FILE
   command  2>>  FILE


(3) 표준 입력 재지정
   command   <   FILE
   command   0<  FILE


(4) 파이프( | )  : 이전 명령어의 실행결과를 다음 명령어의 입력으로 전달하는 것
   command1  |   command2

시그널

시그널 번호     이름     정의      목적
       1            HUP     재설정   프로세스를 종료하지 않고 설정 다시 로드
       2            INT      인터럽트 프로세스에 인터럽트([Ctrl]+[C]) 전달함. 무시가 가능함
       3            QUIT     종료      프로세스에 인터럽트([Ctrl]+[\]) 전달하여 종료.  무시가 가능함.  덤프 발생
       9            KILL      강제종료 프로세스를 강제 종료.  무시 불가능
      15           TERM     종료      프로세스를 정상 종료. 무시가 가능함.
      18           CONT     계속      중지된 프로세스를 재개
      19           STOP      중지      프로세스 중지. 무시 불가능.
      20           TSTP       중지      프로세스 중지([Ctrl]+[Z]). 무시가 가능함.


포그라운드 프로세스(Foreground Process)
 - 프로세스 실행 시 터미널의 제어권을 가져가는 프로세스
 - 프로세스 실행이 종료되면 터미널 제어권을 반납
 - 프로세스 실행 중인 동안 터미널에서 다른 작업을 수행할 수 없음.

백그라운드 프로세스(Background Process)
 - 프로세스 실행 시 백그라운드에서 실행되는 프로세스
 - 제어 터미널을 가지지 않음.
 - 프로세스 실행 후 다른 작업을 수행할 수 있음.

데몬
 - 운영체제 커널이 실행하고 제어하는 백그라운드 프로세스
