Ch03. Cisco IOS

IOS Modes
 - User Mode
 - Privileged Mode
 - Global Configuration Mode
 - Interface Mode
 - Line Mode


Configuration 종류
 - running-config
 - startup-config


네트워크 장비 패스워드
 - enable password
 - enable secret

부팅 과정
1. POST
2. ROM(Bootstrap)
3. IOS(Flash, ROM, TFTP) Load
4. OS Image Loading
5. Startup-Config(NVRAM) -> Running-Config(RAM)
   (만약 Startup-Config가 없으면 Setup-Mode)


네트워크 장비 패스워드 재설정(기존 설정 복구)
1. 장비의 전원을 껐다가 켠다.
2. 부팅 중 [Ctrl]+[Pause Break] 로 ROMMON 모드로 진입한다.
3. Configuration Register 값을 0x2142 로 변경한다.
   rommon > confreg 0x2142
4. ROMMON 모드에서 부팅을 진행한다.
   rommon > boot
5. Startup-Config가 로드되지 않고 부팅이 된다.
6. 기존의 Startup-Config를 불러온다.
   Router# copy startup-config running-config
7. Global Configuration 모드에서 패스워드(시크릿)를 설정한다.
   Router# enable secret cisco
8. Global Configuration 모드에서 Configuration Register 값을 원래의 값인  0x2102  로 복원한다.
   Router(config)# config-register 0x2102
   Router(config)# exit
9. 변경된 설정값을 영구 저장한다.
   Router# copy running-config startup-config
10. 변경된 패스워드를 확인해본다.
11. 네트워크 장비를 재부팅하여 정상 부팅되는 것을 확인한다.
   Router# reload
