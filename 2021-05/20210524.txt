Ch04. Routing

Router
 - 네트워크와 다른 네트워크를 연결하는 L3 장비
 - 목적지까지 데이터를 전달하기 위한 최적의 경로 결정

Routing
 - 데이터를 목적지에 전달하기위해 최적의 경로(Best Route)를 결정하는 과정

Routing 결정 요소
 - 홉수(Hop count)
 - 대역폭(Bandwidth)
 - 부하(Load)
 - 지연(Delay)
 - 신뢰도(Reliability)


Routing
 - Static Routing
    - 네트워크 관리자가 수동으로 라우팅 정보 관리

 - Dynamic Routing
    - 라우팅 프로토콜이 라우팅 정보 관리


Routing
  - Routing Protocol
      라우팅을 수행하는 프로토콜
  - Routed Protocol
      라우팅이 되는 L3의 네트워크 프로토콜
      IP, Novell IPX, AppleTalk

명령어
라우팅 테이블 확인
Router# show ip route

인터페이스 별 IP 주소 간략히 확인
Router# show ip interface brief


Static Routing 구성 명령어
Router(config)# ip route IP_주소 서브넷마스크 NEXT_HOP_IP


Static Default Route 설정
Router(config)# ip route 0.0.0.0 0.0.0.0 NEXT_HOP_IP
